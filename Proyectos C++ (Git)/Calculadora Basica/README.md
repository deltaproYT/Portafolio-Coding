# Calculadora Básica

## Descripción
Calculadora que realiza operaciones básicas: suma, resta, multiplicación y división.

## Características
- Permite al usuario ingresar dos números
- Permite elegir la operación deseada (+, -, *, /)
- Manejo de errores como división por cero

## Tecnologías Usadas
- C++
- Visual Studio Code
