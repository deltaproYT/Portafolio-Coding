# Calculadora de Porcentajes

## Descripción
Calcula qué porcentaje representa un valor parcial sobre un total.

## Características
- Ingreso de valor total y parcial
- Cálculo del porcentaje correspondiente

## Tecnologías Usadas
- C++
- Visual Studio Code
