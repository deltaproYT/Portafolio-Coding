# Conversor Farenheit y Celsius

## Descripción
Convierte temperaturas entre Farenheit y Celsius en ambas direcciones.

## Características
- Permite escoger la dirección de conversión
- Muestra resultados con precisión

## Tecnologías Usadas
- C++
- Visual Studio Code
