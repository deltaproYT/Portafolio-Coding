# Calculadora de Perímetro de Círculo

## Descripción
Calcula el perímetro de un círculo en base al radio ingresado.

## Características
- Uso de constante PI
- Cálculo simple de perímetro usando la fórmula 2 * π * radio

## Tecnologías Usadas
- C++
- Visual Studio Code
